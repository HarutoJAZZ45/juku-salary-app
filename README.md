# Juku Salary App

塾講師向けの給与見込み管理 Web アプリです。勤務した日付をカレンダーから選び、担当コマ、事務作業時間、手当、校舎、交通費を入力すると、締め日ベースの給与見込み、年収上限までの残額、勤務実績、バッジやランキングを確認できます。

## 主な機能

- 月間カレンダーで勤務日を登録、編集、削除
- 複数日を選択して同じ勤務内容を一括登録
- A-G コマ、リーダー/サブリーダー単価、事務作業、手当、交通費、校舎別設定に対応
- 締め日をもとにした給与対象期間の集計
- 年収上限、標準では 103 万円、に対する進捗表示
- 月別収入、コマ数などのグラフ表示
- 連勤、収入、イベント、レベルに応じたバッジ/称号
- 公開設定したユーザー同士の月間/年度ランキング
- 日本語、英語、スペイン語の表示切り替え
- ローカル保存、Firebase 同期、JSON バックアップ/復元

## データ保存

アプリの利用にはログインが必要です。勤務データや設定は、ログイン中のアカウントに紐づけて Cloud Firestore に保存します。

Firebase Authentication の `uid` をキーとして、Firestore の `users/{uid}` に次のデータを保存します。

- `entries`: 日付ごとの勤務データ
- `config`: 時給、交通費、締め日、プロフィールなどの設定
- `uiState`: お知らせ既読、ヘルプ表示済み、称号確認済みなどのUI状態
- `updatedAt`: クラウド同期時刻

ログイン後はクラウド上のデータを正とします。過去バージョンでブラウザ内に残った可能性のある端末内データは読み込まず、削除のみ試みます。

## 給与計算仕様

1 日の給与は `src/utils/calculator.ts` の `calculateDailyTotal` で計算します。

- 通常コマ: `授業時給 * 1.5`
- サブリーダーコマ: `1500 * 1.5`
- リーダーコマ: `2000 * 1.5`
- 所属校舎が平岡で勤務校舎が平岡以外の場合、授業時給を 100 円減額
- 所属校舎が平岡以外で勤務校舎が平岡の場合、授業時給を 100 円増額
- 事務作業: 入力された分数に加え、1 コマあたり 5 分を自動加算
- 連続するコマ間には 10 分を自動加算。ただし B-C 間は昼休み扱いで加算しない
- 事務作業分は `事務時給` で時給換算
- 勤務がある日は勤務手当を加算。平岡は 800 円、それ以外は 400 円
- 任意入力の追加手当を加算
- 交通費ありの場合、日別の交通費、なければ設定値を加算
- 小数点以下は切り捨て

締め日ベースの給与対象期間は `getPeriodRange` で計算します。標準の締め日は 15 日で、たとえば 4 月給与の対象期間は 3 月 16 日から 4 月 15 日です。

## 年収モニター

`TaxMonitor` は支給日ベースで年収を集計します。勤務日、締め日、支払月ラグから支給日を計算し、選択した年の 1 月 1 日から 12 月 31 日までに支給される給与合計を年収として扱います。

標準の上限は `1,030,000` 円です。設定画面から変更できます。

## バッジ、レベル、ランキング

レベルは累計実績から XP を計算します。

- 給与 100 円ごとに 1 XP
- 1 コマごとに 50 XP
- 勤務日 1 日ごとに 50 XP

レベル計算は `XP = 14 * (Level - 1)^2.2` を基準にしています。レベルに応じて称号が解放され、プロフィールで表示称号を選べます。

バッジは主に次の条件で付与されます。

- 3 日、4 日、5 日以上の連勤
- 対象期間の給与見込みが 7 万円、10 万円、13 万円、16 万円以上
- 特定イベント日、例: 2026 年 1 月 2 日または 1 月 3 日の登録

ランキングはプロフィールで公開ランキングを有効にしたユーザーのみ参加します。Firestore の `rankings/{uid}` に月間/年度のコマ数と勤務日数を保存し、ランキング画面で最大 50 件を表示します。

## 主要ディレクトリ

```text
src/
  components/  画面部品と各種モーダル
  contexts/    言語切り替え Context
  data/        お知らせなどの静的データ
  hooks/       認証と給与データ管理
  lib/         Firebase 初期化
  locales/     ja / en / es の翻訳
  types/       TypeScript 型定義
  utils/       給与計算、バッジ、レベル、ランキング処理
```

## 注意点

- `node_modules` と `dist` は生成物なので Git 管理対象外です。
- `.env.local` は秘密情報を含むため Git 管理対象外です。
- ローカル保存はブラウザ単位です。端末をまたいで使う場合はログインしてクラウド同期を利用してください。
