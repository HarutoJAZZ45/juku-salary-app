# Juku Salary App 💰

塾講師のための、シンプルで高機能な給与計算Webアプリです。
日々の勤務をカレンダーに入力するだけで、複雑な給与計算を自動で行い、15日締めの給与見込みをリアルタイムで確認できます。

## ✨ 主な機能

- **📅 直感的なカレンダー入力**
  - タップするだけで勤務日を選択。
  - 担当コマ（A〜G）、勤務校舎、サポート業務の時間などを簡単に入力できます。

- **💴 柔軟な給与計算**
  - **通常授業給**: 所属校舎以外での勤務時には、時給を自動調整（±100円）。
  - **役職給**: リーダー（2,000円/時）、サブリーダー（1,500円/時）の特殊レートに対応。
  - **手当・事務給**: 授業以外の業務時間（分単位）や、その他の手当にも柔軟に対応。

- **🚃 交通費の自動計算**
  - 校舎ごとに交通費を設定可能。
  - 勤務地を選べば、設定した交通費が自動で適用されます。

- **📊 リアルタイム給与明細**
  - **「15日締め・翌月末払い」**のサイクルで自動集計。
  - その月の「振込予定額」がひと目でわかります。

- **💬 フィードバック機能**
  - アプリ内から直接、開発者へ要望やバグ報告を送ることができます。

## 📱 使い方ガイド

### Step 1: 初期設定 ⚙️
右上の「設定（歯車アイコン）」を開き、以下の項目を入力してください。このデータが全ての計算の基準になります。

1. **授業時給**: あなたの基本時給。
2. **サポート時給**: 事務給や最低賃金など。
3. **デフォルトの交通費**: 基本となる交通費。
4. **所属校舎**: あなたのホーム校舎（ここ以外で働くと時給が変わります）。
5. **校舎別交通費**: 各校舎に行くときにかかる交通費を設定しておくと便利です。

### Step 2: 日々の勤務入力 ✏️
カレンダーの日付をタップして、その日の勤務内容を入力します。

1. **勤務校舎**: その日働いた場所を選びます。
2. **担当コマ**: A〜Gのコマを選択します。
   - **役職給を使う場合**: リストの下にある「役職給設定」を開き、リーダー・サブリーダーとして担当したコマを選んでください。
3. **追加業務**: 授業以外で働いた時間（分）を入力します。
4. **交通費**: 必要に応じてチェックを入れます（校舎を選ぶと自動で金額が入ります）。
5. **保存**: 入力したら保存ボタンを押すと、カレンダーに戻ります。

### Step 3: 給与の確認 👀
画面上部のカードに、現在の給与見込みが表示されます。
- 集計期間は「前月16日 〜 当月15日」です。
- 左右の矢印で月を切り替えて、過去や未来の予定も確認できます。
